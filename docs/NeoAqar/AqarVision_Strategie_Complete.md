# 2. Stratégie pour atteindre 10 000 annonces

## Problème principal

Une marketplace immobilière n'a aucune valeur sans inventaire.

La priorité est donc **l'offre**.

------------------------------------------------------------------------

## Étape 1 --- Onboarder les agences

Stratégie terrain :

-   contacter agences physiques
-   démonstration produit
-   création compte gratuite

Objectif :

500 agences.

------------------------------------------------------------------------

## Étape 2 --- Import automatique

Permettre :

-   import Excel
-   import CRM existant
-   import annonces Facebook

------------------------------------------------------------------------

## Étape 3 --- Avantage marketing

Offrir :

-   mini site agence
-   visibilité Google
-   génération automatique des annonces

------------------------------------------------------------------------

## Étape 4 --- Effet réseau

Plus il y a d'annonces :

-   plus les acheteurs viennent
-   plus les agences veulent publier

------------------------------------------------------------------------

# Conclusion

Le succès d'AqarVision dépend de trois facteurs :

1.  Inventaire d'annonces
2.  Expérience utilisateur simple
3.  Valeur ajoutée pour les agences

La technologie seule ne suffit pas :

Il faut combiner produit, distribution et réseau.
